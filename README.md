## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/CyberSaguaros/Programming.git
   ```
   - `cd Programming`

2. **Navigate to your language of choice**
   - `cd Python/Algorithms`

3. **Run a script**
   - `python3 bubble_sort.py`
